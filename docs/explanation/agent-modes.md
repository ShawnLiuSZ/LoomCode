# Agent 模式

> LoomCode 支持四种 Agent 模式，适用于不同场景。

## 模式列表

| 模式 | 说明 | 工具权限 |
|------|------|---------|
| **Build** | 默认模式，完整工具权限 | 读写文件、执行命令 |
| **Plan** | 只读分析模式 | 仅读取文件 |
| **Compose** | 规格驱动开发 | 完整工具权限 |
| **Max** | 并行候选 + 选优 | 完整工具权限 |

## 使用方式

- **Tab 键**：在模式间循环切换
- **命令**：`/build`、`/plan`、`/compose`、`/max`

## 模式特点

### Build 模式
默认模式，Agent 可以读写文件、执行命令、搜索代码。

### Plan 模式
只读分析模式，Agent 只能读取文件，适合代码审查和方案规划。

### Compose 模式
规格驱动开发，Agent 按照规格文档逐步实现功能。

### Max 模式（实验性）
并行运行多个候选 Agent，选择最优结果。
